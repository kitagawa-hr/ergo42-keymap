# Ergo42 Keymap

- `keymap.c` is keymap definition file for qmk firmware.

- `layers.json` is json configuration of keymap.

- `convert.py` converts `layers.json` to `keymap.c`

- `display.py` display `layers.json` as table.

```sh
# layers.json -> keymap.c
pipenv run python convert.py
# display layers.json
pipenv run python diplay.py

# display layers.json and add it to README
pipenv run python diplay.py >> README.md
```
